# Employee Salary Analytics Tool

## 📝 Description
This Python script automates the processing of employee salary data from CSV files. It calculates departmental averages and identifies top earners.

## 🚀 Features
* Processes CSV data using `defaultdict`.
* Generates a timestamped summary report.
* Includes error handling for missing files.

## 🛠️ Requirements
* Python 3.x
* Standard libraries: `csv`, `collections`, `datetime`

## 📖 How to Run
1. Place your `employees.csv` in the root folder.
2. Run the script: `python main.py`
