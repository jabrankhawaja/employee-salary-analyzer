# Employee Salary Analytics Tool 📊

## Description
An automated Python pipeline that reads employee data from CSV files, calculates departmental statistics, and identifies the highest-paid staff.

## Key Skills Demonstrated
* **Modular Programming:** Clean, function-based structure.
* **Data Processing:** Efficient use of `defaultdict` and `csv` modules.
* **Error Handling:** Robust file validation using `try-except`.

## How to Run
1. Ensure `employees.csv` is in the directory.
2. Run: `python main.py`
3. View the generated `summary_YYYY_MM_DD.csv` file.
